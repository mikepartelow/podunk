## Usage

<pre>
1) Edit podunk.cfg.  Add feeds.  Modify podcast download location.
2) ./podunk.rb
</pre>

Run podunk regularly to keep up to date with your subscriptions.  The file www/index.html is updated to reflect current podcast library.
